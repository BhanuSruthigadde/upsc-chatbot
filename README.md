# UPSC Chatbot 🤖

A simple chatbot built using Python and Tkinter to answer UPSC-related queries.

## Features
- Answers eligibility, syllabus, exam pattern
- GUI-based chatbot
- Keyword-based responses

## Tech Used
- Python
- Tkinter

## How to Run
```bash
python chatbot.py
```
